## PHP

Docker example for [php](https://hub.docker.com/_/php)

Extension installation done with: [docker-php-extension-installer](https://github.com/mlocati/docker-php-extension-installer) 

#### Installed extensions:
- bcmath
- calendar
- csv
- exif
- gd
- igbinary
- imagick
- imap
- intl
- ldap
- memcached
- msgpack
- mysqli
- opcache
- pcov
- pdo_mysql
- pdo_pgsql
- pgsql
- readline
- redis
- soap
- zip
